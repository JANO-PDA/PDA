description: Use when creating any new Jetpack Compose UI component. Ensures design system compliance, proper animations, and accessibility.
---
Before creating any Compose component:
1. Read agent_docs/design_system.md for styling rules
2. Read agent_docs/animation_guide.md for motion specs
3. Apply glassmorphism card style from ui/theme/Shape.kt
4. Use colors from ui/theme/Color.kt
5. Add haptic feedback for interactive elements
6. Ensure dark mode compatibility
7. Add content descriptions for accessibility
